# CodeAlpha Data Analytics Project

This repository contains my completed tasks for the CodeAlpha Data Analytics Internship.

## Task 1: Dataset Creation
In this task, a dataset was created manually in CSV format containing quotes and authors.
Due to website accessibility issues, the dataset was prepared as a sample dataset for analysis purposes.
This dataset is used for Exploratory Data Analysis and Sentiment Analysis.

File:
- task1_dataset1.csv

## Task 2: Exploratory Data Analysis (EDA)
In this task, Exploratory Data Analysis was performed on the dataset using Python.
The structure of the data was analyzed and a bar chart was generated to visualize
the number of quotes per author.

File:
- task2_eda.py

Tools Used:
- Python
- Pandas
- Matplotlib

## Task 3: Sentiment Analysis
In this task, sentiment analysis was performed on the dataset using TextBlob.
Each quote was classified into Positive, Negative or Neutral.

Files:
- task3_sentiment.py  
- sentiment_result.csv
